# Striche

A little App to demonstrate AI for the game *Striche* game

## Setup

```bash
npm i
npm start
```
Now the development server listens on [http://localhost:3000](http://localhost:3000)

## Usage

Just drag a line to cross some lines. AI will kick in after 100ms.
